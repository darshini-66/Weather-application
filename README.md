# 🌤 Weather App

A simple and responsive weather application built using HTML, CSS, JavaScript, and Axios. The app allows users to search for any city and view its current temperature using real-time weather data from the OpenWeather API.

## 🚀 Features

- Search weather by city name
- Real-time weather data
- Displays current temperature in Celsius
- Clean and responsive user interface
- Error handling for invalid city names

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Axios
- OpenWeather API

## 📸 Preview

Enter a city name and click **Search** to view the current temperature.
